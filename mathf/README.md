# Mathf
Mathf is a math module for the engine, which provides easy functions to use in game development aswell
in graphics development.

## Implemented

### Vec3, Vec4
- [ ] Plus, Minus, Multiply, Divide
- [x] Dot Product
- [x] Cross Product
- [x] Normalize

### Mat4
- [x] Perspective
- [x] Translate
- [x] Scale
- [ ] RotX, RotY, RotZ
- [ ] LookAt

### Misc
- [x] Noise2D
- [x] Double Plus, Minus, Multiply, Divide
